# 💄 Nykaa Clone

## 📖 About The Project

Nykaa Clone is a front-end recreation of the Nykaa homepage developed using HTML5 and CSS3. The project replicates the visual design and layout of the original website, including navigation menus, promotional banners, product showcase sections, brand highlights, and footer components. It was built to strengthen front-end development skills and gain hands-on experience in creating modern e-commerce user interfaces.

---

## ✨ Features

- Promotional announcement banner
- Navigation bar with search functionality
- Product category navigation menu
- Hero banner section
- Top Brands showcase
- Only at Nykaa section
- Special Curations section
- Beauty & Advice section
- Interactive hover effects
- Footer with multiple navigation links
- Font Awesome icon integration
- CSS-based animations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Animations
- Font Awesome

---

- Working with external resources and icons
- Building real-world website interfaces
